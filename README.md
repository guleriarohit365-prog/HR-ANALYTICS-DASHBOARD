# 📊 HR Analytics Dashboard
Recommended Structure and Order
1. Project Title / Headline
**HR Analytics Dashboard – Employee Attrition & Workforce Insights**
An interactive Power BI dashboard designed to analyze employee data and uncover workforce patterns through attrition analysis, salary trends, age distribution, education background, and job role insights.

 2. Short Description / Purpose

The HR Analytics Dashboard is a visually interactive and analytical Power BI report developed to help organizations monitor employee performance and understand attrition behavior. It enables HR teams to make data-driven decisions by identifying trends across departments, demographics, salary groups, and employee tenure.

3. Tech Stack

The dashboard was built using the following tools and technologies:

- 📊 **Power BI Desktop** – Dashboard development and visualization  
- 🔄 **Power Query** – Data cleaning and transformation  
- 🧮 **DAX (Data Analysis Expressions)** – Calculated columns, measures 
- 📁 **CSV / Excel Dataset** – Data source and management  
- 📐 **Data Modeling** – Relationship building and report optimization
